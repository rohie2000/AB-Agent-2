# Vorlagen

In diesen Ordner kommen die echten Arbeitsblatt-Vorlagen.

Moegliche Formate:

- `docx`
- `html`
- `pdf` als Referenz
- lokale Schriftdateien, falls fuer das Rendering erlaubt

Empfehlung:

- lege hier die Hauptvorlage ab
- dokumentiere daneben kurz, welche Platzhalter spaeter befuellt werden sollen
- notiere auch, wie die Schriftart "OpenSan" technisch eingebunden wird

Aktueller Stand im Projekt:

- `AB-Vorlage-Blank.pdf`
- `AB-Vorlage-mit-Arbeitsanweisungen.pdf`

Diese beiden PDFs dienen momentan als visuelle Referenz fuer den HTML-Renderer unter
`scripts/render_worksheets.py`.
